About

Purpose & Technology

Both projects are fully independent — no shared backend, no merged logic.

📊 FraudGuard

Detects anomalous financial transactions using unsupervised machine learning. Designed for analysts who need to upload raw transaction data and immediately identify high-risk patterns without labelled training data.

  • Flask REST API backend
  • Ensemble of 3 unsupervised ML models
  • SQLite with blockchain-style audit log
  • Real-time monitoring thread
  • Power BI CSV/JSON export

🛡️ SpamShield

Classifies emails as spam or ham using supervised machine learning trained on labelled datasets. Supports single message input, file upload, and bulk CSV processing with per-row predictions.

  • Django MVC backend
  • Voting ensemble (RF + LR + SVM)
  • TF-IDF + hand-crafted feature engineering
  • Alert system with email/webhook notifications
  • Model retraining endpoint

Getting Started

How to Run Both Projects

Run each project in a separate terminal window. Both must be running to use the preview buttons.

1

Run FraudGuard (Port 5000)

Open a terminal, navigate to the pbi card folder and start Flask.

cd "pbi card" pip install -r requirements.txt python app.py --port 5000
2

Run SpamShield (Port 5001)

Open a second terminal, navigate to the spam detection folder and start Django.

cd Spam-Detection-Classifier-main\Spam-Detection-Classifier-main\spam_detection pip install -r requirements.txt python manage.py runserver 5001 --noreload
3

Open This Page

Simply open this index.html file directly in your browser — no server needed.

cd launcher python serve.py

Then open: http://localhost:8000

4

One-Click Launch (Windows)

Use the batch script to start all apps at once from the project root.

run_all.bat

Opens both servers and this page automatically.
